[ 스레드 제어함수 ]
-> 스레드 제어 함수는 스레드의 생명 주기(Life Cycle)를 관리하고, 스레드 간의 상호작용을 조절하는 데 사용

1. start() : 스레드 실행 시작
2. join() : 스레드 종료 대기
3. sleep() : 스레드 일시 정지
4. yield() : 스레드 실행 양보
5. interrupt : 스레드 인터럽트
6. isInterrupted() : 스레드 인터럽트 상태 확인

7. 동기화 관련 함수
   - wait() : 스레드를 잠시 잠재우는 것
   - notify() : 특정 스레드를 깨우는 것( 하나의 스레드만 적용될 경우, 상관없지만 어느놈을 깨우는지 보장되지 않아 권장사용 X )
   - notifyAll() : 잠들어 있는 모든 스레드를 일괄로 깨우는 것
